This is my first Power BI project, focused on analyzing a global financial dataset. The goal of this project was to transform raw sales data into an interactive dashboard that provides actionable insights into product performance, regional sales, and profitability.

## üìÅ Dataset Description
The analysis is based on a sales dataset encompassing:
* **Products:** Carretera, Paseo, Velo, VTT, Amarilla, and Montana.
* **Countries:** Germany, Canada, France, Mexico, and the USA.
* **Segments:** Government, Midmarket, Channel Partners, Small Business, and Enterprise.
* **Key Metrics:** Units Sold, Manufacturing Price, Sale Price, Gross Sales, Discounts, COGS, and Profit.

## üöÄ Key Features of the Dashboard
* **Executive Summary:** High-level cards showing Total Sales, Total Profit, and Total Units Sold.
* **Product Analysis:** A breakdown of which products drive the most revenue vs. which are most profitable.
* **Geographic Insights:** A map visualization showing sales distribution across North America and Europe.
* **Discount Impact:** Analysis of how "None" vs. "Low" discount bands affect the bottom line.
* **Temporal Trends:** A line chart showing sales fluctuations over the project's timeframe.
